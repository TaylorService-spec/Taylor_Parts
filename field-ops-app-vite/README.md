# Field Ops (Vite)

Vite/React build of the Field Ops app — Jobs, Technicians, Dispatch, and
Control Tower views backed by Firestore. Replaces the Babel-in-browser
version in `../field-ops-app/` with a real build pipeline and native ES
modules.

## Develop

```
npm install
npm run dev
```

## Build

```
npm run build
```
